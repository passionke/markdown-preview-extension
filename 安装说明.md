# Markdown Preview 扩展 - 安装到 Cursor

## 方式一：用脚本（需管理员）

1. **右键 PowerShell → 以管理员身份运行**
2. 执行：
   ```powershell
   cd "d:\刀枫\auto"
   .\install_node_and_extension.ps1
   ```
3. 完成后**重启 Cursor**

---

## 方式二：按你给的步骤手动装

**1. 以管理员身份打开 PowerShell，安装 Chocolatey：**
```powershell
powershell -c "irm https://community.chocolatey.org/install.ps1|iex"
```

**2. 安装 Node.js：**
```powershell
choco install nodejs --version="24.13.1" -y
```

**3. 关闭并重新打开终端/Cursor，验证：**
```powershell
node -v   # 应显示 v24.13.1
npm -v    # 应显示 11.x
```

**4. 编译并打包扩展：**
```powershell
cd "d:\刀枫\auto\markdown-preview-extension-temp"
npm install
npm run compile
npx vsce package --no-dependencies
```

**5. 在 Cursor 里安装 VSIX：**
- `Ctrl+Shift+P` → 输入 **Extensions: Install from VSIX**
- 选择当前目录下的 `markdown-preview-ext-0.1.0.vsix`
- 重启 Cursor
